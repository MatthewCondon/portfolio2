Scaled 2

The purpose of this project was to make another scaled item. However, this project was determined by a single variable rather than the canvas size. Changing the x variable will change the size of both faces with respect to the proportions. Moreover, clicking on the screen will display a different face.

Like Scaled 1 , the hardest part of this project was determining the proportions for the faces.

If I were to refactor this project, I would want to add more face designs.

I hope you like this Picasso-inspired project!
