Solar System

I made this project at the start of the year to test an array of my abilities. For starters, this project was new in that it used sin and cos to make the orbits of the planets. There was also an array of stars randomly placed in the background. Lastly, all the planets orbit around the sun at different speeds.

The hardest part was making the planets follow a circular orbit at different speeds. I overcame this issue by assigning different values to sin and cos.

If this project was refactored, I would make it scale.
