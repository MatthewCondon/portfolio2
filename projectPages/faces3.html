Faces 3

The goal of this project was to test the skills we practiced in Faces 1 and Faces 2 . Essentially, there is a double for-loop that determines the location of the faces. Moreover, when the mouse is clicked, the design of the faces change. Finally, the eyes change colors and the pupils move around.

The hardest part of this project was making the shapes of the mouths change with the faces. Essentially, changing the mouth shape was not dependent upon user input. This was done with a series of random variables.

If I were to refactor this project, I would want to make more faces of different sizes.

I hope you find this project enjoyable!
